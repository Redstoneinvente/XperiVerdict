# XperiVerdict

**XperiVerdict** is a community-driven Sony Xperia firmware tracker that aggregates official updates, security patches, OS upgrades, and real-world stability intelligence in one place.

No marketing language. No speculation. Just firmware data, clearly and responsibly presented.

---

## What Problem Does This Solve?

Sony firmware updates are often:

- Distributed inconsistently across regions  
- Poorly documented  
- Accompanied by vague or incomplete changelogs  
- Difficult to compare across different models  

As a result, users are frequently left asking:

- Is this update stable?  
- What actually changed?  
- Is it worth installing right now?
