Guide

How to Update Sony Xperia Software

Safe methods to update your Xperia: OTA updates, Xperia Companion on PC/Mac, and fixes when the update is not showing.

Preparation

Before you update

1

Charge the phone

Keep at least 50% battery (70%+ recommended) and plug in during install.

2

Use reliable Wi-Fi

Large updates fail more often on unstable networks.

3

Back up what matters

Photos + critical files. OTA usually preserves data, but don’t gamble.

4

Free storage

Keep 5–10 GB available. Low space is a common update failure cause.

Tip: If you care about stability, check your build on XperiVerdict and scan early community intel before updating.

Method 1

Update over-the-air (OTA)

1

Open System Update

Settings → System → System update (wording varies by Android version).

2

Check and download

Tap Check for update, then download if available.

3

Install safely

Tap Install. Keep it charging. Don’t reboot mid-install.

Warning: If the phone reboots or dies mid-install, you can end up in a boot loop. Keep it charged.
Method 2

Update via Xperia Companion (PC/Mac)

Use this if OTA is failing or the update is not appearing. It’s slower, but more reliable.

1

Install Xperia Companion

Install the Sony tool on Windows or macOS.

2

Connect via USB

Use a stable cable and connect directly (avoid flaky hubs).

3

Run Software update

Select Software update and follow prompts without disconnecting.

Warning: “Repair” flows may wipe data depending on the option used. Back up first when possible.
Troubleshooting

Update not showing

1

Staged rollout is normal

Updates roll out by region, model code, and sometimes carrier.

2

Try basic fixes

Reboot, disable VPN, and check again on another network.

3

Use Xperia Companion

Companion sometimes detects updates before your OTA feed catches up.

Post-update

After updating (stability checklist)

1

Let it settle

Keep it charging for 15–30 minutes after first boot (background optimisation).

2

Restart once if needed

If heat/battery drain is odd on day one, restart once before panicking.

3

Update apps

Play Store updates reduce compatibility issues after OS/security changes.
